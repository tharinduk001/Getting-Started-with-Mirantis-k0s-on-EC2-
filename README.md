# Mirantis

Mirantis is a cloud-native software company that helps organizations adopt and operate modern infrastructure and application platforms. The company focuses on delivering production-grade Kubernetes, container, and edge solutions that simplify cluster lifecycle, security, and operations across cloud and on-premises environments.

**What Mirantis does**

- Provides Kubernetes distributions and managed services for production environments.
- Develops lightweight Kubernetes distributions and tooling for edge and IoT (e.g., k0s).
- Offers container runtime, lifecycle tooling, and professional services for cloud-native transformation.

**Key offerings & projects**

- Mirantis Kubernetes Engine (MKE) — enterprise-grade Kubernetes distribution.
- k0s — a lightweight, single-binary Kubernetes distribution ideal for edge and resource-constrained environments.
- Container runtimes, CI/CD integrations, and consulting services to help teams run cloud-native workloads reliably.
